# Retail_Demand_Forecasting

DEMAND FORECASTING FOR RETAIL INVENTORY OPTIMIZATION

Problem Statement:

Retailers face challenges balancing stock levels to meet demand without overstocking. Accurate sales forecasts enable optimal inventory management. Using time series data and contextual features, ML models can predict future demand. Students will work with a real retail demand forecasting dataset.
